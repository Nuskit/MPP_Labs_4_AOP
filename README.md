# MPP_Labs_4_AOP
Write a logger, using Mono.Cecil.

Two main project in solution.
1)Logger
Log all methods in class.
To add a logger, you need to add it as an attribute and give one-time fee for the Interpetator IL executable. To pass on all nested classes in the marked for logging class Additional add the -r options.
2)WeakDelegate
Change Delegate to WeakDelegate(Delegate.Target set WeakReference).
To add a logger, you need to add it as an attribute and give one-time fee for the Interpetator IL executable.
